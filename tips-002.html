Prompting

Prompt yang Efektif

Kualitas output ChatGPT sangat bergantung pada cara kamu nulis prompt. Ini yang perlu diperhatikan — plus cara setup Custom Instructions agar tidak perlu explain ulang setiap sesi.

✅Do This
  • Berikan konteks: siapa kamu, untuk apa, output seperti apa
  • Tentukan format output — bullet, paragraf, tabel, kode
  • Sebutkan panjang yang diinginkan — "singkat", "detail", "max 200 kata"
  • Gunakan role: "Kamu adalah [ahli X], bantu saya..."
  • Iterasi bertahap — mulai dari draft, lalu refine
  • Setup Custom Instructions sekali untuk semua sesi
❌Don't Do This
  • Prompt 1 kalimat tanpa konteks untuk task kompleks
  • Berharap ChatGPT "tebak" maksud kamu
  • Minta 5 hal sekaligus dalam 1 prompt
  • Pakai model reasoning (o1/o3) untuk chat biasa — boros & lambat
  • Lanjut di chat yang sudah sangat panjang tanpa summary
  • Ignore Custom Instructions — harus setup dari awal terus
1
Setup Custom Instructions Sekarang
Pergi ke Settings → Personalization → Custom Instructions. Isi "Tentang kamu" (profesi, bahasa, gaya komunikasi) dan "Cara ChatGPT harus respond" (format, panjang, tone). Berlaku untuk semua chat baru — tidak perlu explain ulang setiap sesi.
⚡ One-time Setup
2
Gunakan Kerangka C-T-F-O
Context (siapa & situasinya) + Task (apa yang mau dilakukan) + Format (output seperti apa) + Output length (panjang jawaban). Contoh: "Saya seorang marketer [C], buatkan caption Instagram untuk produk skincare [T], dalam format bullet 3 poin [F], max 100 kata [O]."
🎯 Prompt Framework
3
Pakai GPT-5 sebagai Default Baru
GPT-5 menggabungkan kecepatan GPT-4o dengan kemampuan reasoning o-series dalam satu model. Untuk sebagian besar task sehari-hari, GPT-5 adalah pilihan terbaik — tidak perlu lagi bingung pilih antara GPT-4o vs o1.
🌟 Model Tips
4
Simpan Prompt Template yang Sering Dipakai
Buat dokumen kecil berisi prompt template untuk task yang sering kamu lakukan — nulis email, summarize artikel, draft konten, dll. Tinggal copy-paste dan ganti variabelnya. Hemat waktu dan hasilnya konsisten.
📋 Productivity
Memory & Context

Kelola Memory dengan Bijak

ChatGPT punya fitur Memory yang bisa menyimpan informasi tentang kamu lintas sesi. Kalau dikelola dengan baik, ini sangat powerful — tapi kalau dibiarkan, bisa jadi berantakan.

🔍
Cek & Audit Memory Secara Berkala
Buka Settings → Personalization → Manage Memory. Lihat apa yang disimpan ChatGPT tentang kamu. Hapus yang sudah tidak relevan atau salah — memory lama bisa mempengaruhi jawaban baru.
💾
Minta ChatGPT Simpan Info Penting
Kamu bisa bilang "Ingat bahwa saya seorang [profesi] yang sering butuh [X]" — ChatGPT akan menyimpannya ke memory. Berguna untuk preferensi atau konteks yang selalu berulang.
🚫
Nonaktifkan Memory untuk Chat Sensitif
Kalau mau diskusi topik yang tidak ingin disimpan, mulai chat baru dengan Memory dimatikan sementara — atau pakai mode Temporary Chat (tidak menyimpan apapun ke history dan memory).
🔄
Reset Chat untuk Task Baru yang Tidak Berkaitan
Sama seperti prinsip di Claude Tips — mulai chat baru untuk task yang tidak berkaitan. Context lama bisa "mengkontaminasi" jawaban untuk task baru yang berbeda topik.
📝
Gunakan Summary di Awal Chat Panjang
Untuk project yang butuh banyak sesi, buat "summary doc" yang berisi progress dan context penting. Paste di awal setiap chat baru — ChatGPT langsung paham tanpa baca ribuan pesan sebelumnya.
🎯 Pro Tip
Buat satu chat khusus sebagai "notepad" untuk masing-masing project besar. Semua diskusi, keputusan, dan output tersimpan di satu tempat — mudah dicari kembali dan tidak campur dengan chat lain.

Tips dalam panduan ini berdasarkan pengalaman praktis penggunaan ChatGPT dan bukan panduan resmi dari OpenAI. Perbandingan model bersifat estimasi umum — performa aktual bisa berbeda tergantung task spesifik dan versi model terkini.

Model ChatGPT terus berkembang cepat. GPT-5 dan model o-series masih dalam pengembangan aktif — fitur, harga, dan akses bisa berubah sewaktu-waktu. Selalu cek platform.openai.com untuk info model terkini.

Prinsip-prinsip di sini (prompt yang jelas, kelola context, pilih tool yang tepat) bersifat universal — berlaku juga untuk AI tools lain seperti Claude, Gemini, dan lainnya.
